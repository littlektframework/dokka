useOriginalSize

var useOriginalSize: Boolean = false

If true, then the internal size calculations with use the TextureSlice.originalWidth&TextureSlice.originalHeight for its sizing instead of the trimmed width & height. This is useful if a TextureSlice requires the empty margin around it.
